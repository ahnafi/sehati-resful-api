# Sehati

restful api project for sehati app

## response api

- response 200 success

```json
{
    "data": {
        "...": "..."
    }
}
```

- response error 400 Bad Request

```json
{
    "errors": {
        "field": [
            "..."
        ]
    }
}
```

- response error 401 Unauthorized

```json
{
    "errors": {
        "message": [
            "Unauthorized"
        ]
    }
}
```
